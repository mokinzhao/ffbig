## 项目介绍

- 项目地址：<http://www.ffbig.cn/>
  项目来源于博主自己的职业生涯技能总结和梳理，内容在不断充实中欢迎 fork。
  全栈大前端系列总共分为 3 期：
- 1 期梳理基础知识和常用技术栈
- 2 期学习跨端融合技术认识"大前端"
- 3 期结合服务端技术通晓 "全栈"
  希望能给有需要的朋友提供学习及参考，如觉得内容不错可 start 表示鼓励，谢谢！🙏
