## 概念

- Stack:先入后出；添加、删除皆为 O(1),查询为O(n)

- Queue: 先入先出；添加、删除皆为O(1),查询为O(n)

## 判断有效括号

```js
const isValid= function(s){

    let map={
        '{':'}',
        '(':')',
        '[':']'
    }
    let stack=[]
    for(let i=0;i<s.length;i++){
        if(map[s[i]]){
            stack.push(s[i])
        }else if(s[i] !== map[stack.pop()]){
            return false
        }
    }
    return stack.length === 0
}

```
