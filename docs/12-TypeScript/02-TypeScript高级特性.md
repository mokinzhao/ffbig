## 泛型

### 特性

1. 函数和类可以轻松地支持多钟类型，增强程序的扩展性
2. 不必写多条函数重载，冗长的联合类型声明，增强代码可读性
3. 灵活控制类型之间的约束
4. 泛型可以写多个
