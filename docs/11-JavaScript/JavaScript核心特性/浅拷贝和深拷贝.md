---
title: 浅拷贝和深拷贝
---

## 浅拷贝

对于对象或数组类型，当我们将 a 赋值给 b，然后更改 b 中的属性，a 也会随着变化。

也就是说，a 和 b 指向了同一块堆内存，所以修改其中任意的值，另一个值都会随之变化，这就是浅拷贝。

## 深拷贝

那么相应的，如果给 b 放到新的内存中，将 a 的各个属性都复制到新内存里，就是深拷贝。

也就是说，当 b 中的属性有变化的时候，a 内的属性不会发生变化。
