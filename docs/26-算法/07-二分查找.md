## 概念

- 前提条件

1. 目标函数单调性（单调递增或递减）

2. 存在上下界

3. 能够通过索引访问

- 代码模版

```js
let left ,right =0

const end= len(array)-1

while(left<=right){
    const mid = (left+right)/2
    if(array[mid]==target){
        // find the target
        break or return result
    }
    else if(array[mid]<target){
        left= mid+1
    }
    else {
        right =mid-1
    }
      
}
```
